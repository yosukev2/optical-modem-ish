# Claude Code ガイドライン

このファイルはClaude Codeがリポジトリで作業する際のルールを定義する。

## PR作成時のルール

- `.github/pull_request_template.md` のフォーマットに必ず従うこと
- PRを作成する前にテンプレートを読み、全セクションを埋めること
- テンプレートのチェックリスト項目は該当するものにチェックを入れる
